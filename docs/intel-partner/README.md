# SWGI Intel Partner Readiness

This directory maps SWGI to Intel partner positioning and customer deployment
readiness.

## Documents

- `ARCHITECTURE.md`: Intel-aligned SWGI architecture for Kubernetes, edge, and
  confidential workload scenarios.
- `DEPLOYMENT_GUIDE.md`: customer deployment guidance for Intel-based
  infrastructure.
- `VALIDATION_CHECKLIST.md`: validation evidence for Intel partner and customer
  reviews.

## Scope

Intel partner readiness focuses on:

- Kubernetes governance on Intel Xeon infrastructure.
- Optional Intel TDX / confidential computing alignment.
- Hardware and workload evidence captured through SWGI Trust Receipts.
- Enterprise deployment guidance for cloud, private cloud, and edge.

It does not replace Google Cloud Marketplace packaging. The Google deployment
package remains in `../../marketplace/google`.
